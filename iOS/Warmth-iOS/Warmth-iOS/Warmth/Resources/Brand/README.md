# Warmth Brand Assets

Sun-runner logo: **exact original artwork**, recolored to the Warmth app palette.

## Color mapping

| Original | Warmth token | Hex |
|----------|--------------|-----|
| Black background | Ember gradient | `#FF2D1A` → `#FF5A2C` → `#FF9A3D` |
| White character | Warm White | `#FBF8F5` |
| Face details (eyes/mouth) | Ink | `#0B0B0C` |

Geometry is unchanged. Anti-aliased edges are preserved.

## Regenerate

```bash
cd iOS/Warmth-iOS
.venv-brand/bin/python3 scripts/generate_brand_assets.py
```
